# 입고 알리미 (텔레그램) — 소니 / 후지필름

공식스토어의 상품이 **품절 → 판매중**으로 바뀌는 순간을 감지해서
**텔레그램으로 즉시 알림**을 보내주는 프로그램입니다.

- 지원: **소니코리아 공식스토어**, **후지필름코리아 공식스토어**
- 추가 프로그램 설치 불필요 (파이썬 표준 기능만 사용)

## 두 가지 실행 방법

| 방법 | 특징 | 설명서 |
|------|------|--------|
| ☁️ **클라우드 (추천, 현재 사용 중)** | PC 꺼도 24시간 감시, 5분 간격, 무료 | **[README_클라우드.md](README_클라우드.md)** |
| 💻 로컬 PC | PC 켜져 있는 동안만, 1분 간격 가능 | 이 문서 아래 내용 |

> 클라우드 방식이 이미 설정되어 돌아가고 있다면 아래는 안 읽어도 됩니다.
> 아래는 로컬 PC에서 (추가로/대신) 돌리고 싶을 때의 안내입니다.

---

## 1. 준비물

- **파이썬** (이미 설치되어 있습니다: Python 3.13)
- **텔레그램 앱** (휴대폰 또는 PC)

---

## 2. 텔레그램 봇 만들기 (최초 1회)

1. 텔레그램에서 **@BotFather** 를 검색해 대화를 엽니다.
2. `/newbot` 입력 → 봇 이름과 사용자명(끝이 `bot`)을 정합니다.
3. BotFather가 주는 **봇 토큰**(예: `8012345678:AAH...xyz`)을 복사합니다.
4. 방금 만든 **내 봇과의 대화창을 열고 아무 메시지**(예: `안녕`)나 한 번 보냅니다. ← 꼭 해야 함!

---

## 3. 설정하기

### (1) 봇 토큰 넣기
`config.json` 파일을 메모장으로 열어 `telegram_bot_token` 에 위에서 복사한 토큰을 붙여넣습니다.

### (2) 내 chat_id 알아내기
- `챗ID_확인.bat` 을 더블클릭합니다. (또는 텔레그램에서 **@userinfobot** 에게 `/start`)
- 나오는 숫자를 `config.json` 의 `telegram_chat_id` 에 넣습니다.

### (3) 저장
`config.json` 예시 (토큰/챗ID와 동작 설정만 들어갑니다):

```json
{
  "telegram_bot_token": "8012345678:AAH...xyz",
  "telegram_chat_id": "123456789",
  "check_interval_seconds": 60,
  "repeat_alert_minutes": 10,
  "send_startup_message": true
}
```

**감시할 상품 목록은 `products.json`** 에 있습니다 (클라우드와 공용).
상품 추가 방법은 [README_클라우드.md](README_클라우드.md)의 "감시 상품 추가/변경" 참고.

---

## 4. 실행

- **`실행.bat`** 을 더블클릭하면 감시가 시작됩니다.
- 시작하면 텔레그램으로 "🟢 시작" 메시지가 옵니다. (오면 설정 성공!)
- 상품이 입고되면 **🚨 입고 알림!** 메시지와 구매 링크가 옵니다.
- 종료하려면 실행 창에서 **Ctrl + C**.

> ⚠️ 로컬 방식은 이 프로그램이 켜져 있는 동안만 감시합니다. **PC가 켜져 있어야** 합니다.
> PC를 끄고도 감시하려면 클라우드 방식([README_클라우드.md](README_클라우드.md))을 쓰세요.

---

## 5. 설정값 설명 (config.json)

| 항목 | 설명 |
|------|------|
| `check_interval_seconds` | 재고 확인 주기(초). 기본 60초. 30초 미만은 차단 위험이 있어 자동으로 30초로 올립니다 |
| `repeat_alert_minutes` | 입고된 상태가 계속되면 몇 분마다 다시 알릴지. 놓치지 않도록 반복 알림 |
| `send_startup_message` | 시작할 때 확인용 메시지를 보낼지 여부 |

## 파일 설명

| 파일 | 용도 |
|------|------|
| `실행.bat` | 로컬 감시 시작 (더블클릭) |
| `챗ID_확인.bat` | 내 텔레그램 chat_id 확인 |
| `config.json` | 로컬 전용 설정 (토큰/챗ID/주기) — git에 올라가지 않음 |
| `products.json` | 감시할 상품 목록 (클라우드와 공용) |
| `monitor.py` | 감지/알림 핵심 로직 (클라우드·로컬 공용) |
| `sony_stock_alert.py` | 로컬 실행기 (monitor.py 를 반복 호출) |
| `find_chat_id.py` | chat_id 확인 도우미 |
| `state_local.json` | 자동 생성. 로컬 실행의 마지막 재고 상태 (지워도 됨) |
| `.github/workflows/` | 클라우드 자동 실행 설정 (감시 + 저장소 유지) |
