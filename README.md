# 소니 RX100M7 입고 알리미 (텔레그램)

소니코리아 공식스토어의 상품이 **품절 → 판매중**으로 바뀌는 순간을 감지해서
**텔레그램으로 즉시 알림**을 보내주는 프로그램입니다.

- 추가 프로그램 설치 불필요 (파이썬 표준 기능만 사용)
- 소니 스토어 공개 재고 API를 직접 조회 → 빠르고 정확
- 기본 1분마다 확인

---

## 1. 준비물

- **파이썬** (이미 설치되어 있습니다: Python 3.13)
- **텔레그램 앱** (휴대폰 또는 PC)

---

## 2. 텔레그램 봇 만들기 (최초 1회)

1. 텔레그램에서 **@BotFather** 를 검색해 대화를 엽니다.
2. `/newbot` 입력 → 봇 이름과 사용자명(끝이 `bot`)을 정합니다.
3. BotFather가 주는 **봇 토큰**(예: `8012345678:AAH...xyz`)을 복사합니다.
4. 방금 만든 **내 봇과의 대화창을 열고 아무 메시지**(예: `안녕`)나 한 번 보냅니다. ← 꼭 해야 함!

---

## 3. 설정하기

### (1) 봇 토큰 넣기
`config.json` 파일을 메모장으로 열어 `telegram_bot_token` 에 위에서 복사한 토큰을 붙여넣습니다.

### (2) 내 chat_id 알아내기
- `챗ID_확인.bat` 을 더블클릭합니다.
- 화면에 나오는 `chat_id = 숫자` 값을 복사해서 `config.json` 의 `telegram_chat_id` 에 넣습니다.

### (3) 저장
`config.json` 을 저장하면 끝입니다. 예시:

```json
{
  "telegram_bot_token": "8012345678:AAH...xyz",
  "telegram_chat_id": "123456789",
  "products": [
    {
      "name": "소니 RX100 VII (DSC-RX100M7)",
      "product_no": 102263765,
      "page_url": "https://store.sony.co.kr/product-view/102263765"
    }
  ],
  "check_interval_seconds": 60,
  "repeat_alert_minutes": 10,
  "send_startup_message": true
}
```

---

## 4. 실행

- **`실행.bat`** 을 더블클릭하면 감시가 시작됩니다.
- 시작하면 텔레그램으로 "🟢 시작" 메시지가 옵니다. (오면 설정 성공!)
- 상품이 입고되면 **🚨 입고 알림!** 메시지와 구매 링크가 옵니다.
- 종료하려면 실행 창에서 **Ctrl + C**.

> ⚠️ 이 프로그램이 켜져 있는 동안만 감시합니다. **PC가 켜져 있어야** 알림을 받을 수 있습니다.
> 컴퓨터를 끄면 감시도 멈춥니다. (24시간 감시를 원하면 아래 "항상 켜두기" 참고)

---

## 5. 설정값 설명

| 항목 | 설명 |
|------|------|
| `check_interval_seconds` | 재고 확인 주기(초). 기본 60초. 너무 짧게(예: 5초) 하면 차단될 수 있어 30초 이상 권장 |
| `repeat_alert_minutes` | 입고된 상태가 계속되면 몇 분마다 다시 알릴지. 놓치지 않도록 반복 알림 |
| `send_startup_message` | 시작할 때 확인용 메시지를 보낼지 여부 |
| `products` | 감시할 상품 목록. 여러 개 추가 가능 |

### 다른 상품도 감시하려면
소니 스토어 상품 페이지 주소가 `store.sony.co.kr/product-view/**숫자**` 형태입니다.
그 **숫자**를 `product_no` 에 넣고 `products` 배열에 항목을 추가하면 됩니다.

```json
"products": [
  { "name": "RX100M7", "product_no": 102263765, "page_url": "https://store.sony.co.kr/product-view/102263765" },
  { "name": "다른상품", "product_no": 100000000, "page_url": "https://store.sony.co.kr/product-view/100000000" }
]
```

---

## 6. (선택) 항상 켜두기 — 24시간 감시

PC를 계속 켜둘 수 있다면 **Windows 작업 스케줄러**로 부팅 시 자동 실행되게 할 수 있습니다.
원하시면 설정 방법을 알려드리겠습니다. (또는 컴퓨터 절전 모드를 꺼두세요.)
